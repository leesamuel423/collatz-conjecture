About the Collatz Conjecture

The Collatz conjecture is one of the most famous unsolved problems in mathematics. The conjecture asks whether repeating two simple arithmetic operations will eventually transform every positive integer into 1.

Named after Lothar Collatz, who introduced the idea in 1937, the problem is also known as the 3n+1 problem, the Syracuse problem, or the hailstone sequence.

The sequence is defined as follows: start with any positive integer n. Then each term is obtained from the previous term according to these rules: 

if n is even:
    n = n / 2
else:
    n = 3 × n + 1

Despite its simple appearance, the Collatz conjecture has resisted all attempts to prove it for over 80 years. Paul Erdős said about the Collatz conjecture: "Mathematics may not be ready for such problems."

  • The conjecture has been verified for all starting values up to 268 (approximately 2.95 × 1020)
  • For the number 27, the sequence reaches a peak of 9,232 before eventually descending to 1
  • The sequence for most starting numbers contains more steps going up than down, yet still always reaches 1
  • The sequence creates patterns called "hailstone sequences" because the values tend to go up and down, like hailstones in a cloud
